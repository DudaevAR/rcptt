Tags

To improve management and navigation for big test projects, any test can have tags assigned to it. Tags are just short text labels, allowing to quickly find all scenarios containing tags. Tags can be entered directly into Tags field in Test editor (separated by comma), or added via Tag select dialog (green "plus" icon next to Tags field in Test editor).

There are three ways to perform search by tag:

  1. Type tag name into Q7 explorer filter
  2. Use Q7 search dialog
  3. Use tags view
